# PROJECT FOUNDATION

# DEVI – Domestic Violence and Crimes Against Women Analysis using Machine Learning & Deep Learning

## Project Overview

DEVI is a data-driven machine learning and deep learning project developed to analyze historical crime records related to women in India. The project aims to study long-term crime patterns, identify significant trends, and build predictive models that can assist researchers, policymakers, and public safety organizations in understanding crimes against women through data analytics.

The project combines data preprocessing, exploratory data analysis (EDA), machine learning, and deep learning techniques to extract meaningful insights from publicly available crime datasets.

---

## Problem Statement

Crimes against women remain a major social challenge in India. Despite the availability of large amounts of crime data, it is often underutilized for predictive analysis and informed decision-making. Traditional statistical methods provide descriptive insights but may fail to uncover complex patterns across regions and years.

This project addresses this gap by applying machine learning and deep learning techniques to historical crime datasets, enabling the discovery of hidden patterns and supporting data-driven analysis of crimes against women.

---

## Objectives

The primary objectives of this project are:

- Analyze historical crimes against women in India.
- Clean and preprocess multiple datasets.
- Perform Exploratory Data Analysis (EDA).
- Study yearly and state-wise crime trends.
- Identify important factors influencing crime patterns.
- Develop and compare multiple Machine Learning models.
- Build a Deep Learning model for predictive analysis.
- Evaluate model performance using standard evaluation metrics.
- Generate meaningful visualizations and analytical insights.

---

## Motivation

The increasing availability of crime data presents an opportunity to use artificial intelligence for social impact. By leveraging machine learning and deep learning, this project seeks to transform raw crime statistics into actionable insights that may support academic research and future policy discussions.

The scope of DEVI includes:

## Dataset Collection

This project utilizes publicly available datasets containing information on crimes against women across different states and years in India.

The datasets were collected from publicly accessible sources derived from NCRB (National Crime Records Bureau) reports and cleaned datasets prepared for data analysis.

---

## Datasets Used

### Dataset 1
**Crime against Women (2001–2022)**

Description:
- National-level crime statistics
- Covers the period from **2001 to 2022**
- Contains yearly crime counts categorized by crime head

Main Features:
- Crime Head
- Annual crime statistics (2001–2022)

---

### Dataset 2
**Crimes_On_Women_Data**

Description:
A state-wise dataset containing crime statistics across multiple years.

Features include:

- State
- Year
- Rape
- Kidnapping & Abduction (K&A)
- Dowry Deaths (DD)
- Assault against Women (AoW)
- Assault of Modesty (AoM)
- Domestic Violence (DV)
- Human Trafficking (WT/Trafficking)
- Region

---

### Dataset 3
**CrimesOnWomenData**

Description:
A cleaned and standardized version of the state-wise crime dataset containing descriptive feature names for improved readability and analysis.

Features include:

- State
- Year
- Rape
- Kidnapping & Assault
- Dowry Deaths
- Assault against Women
- Assault of Modesty
- Domestic Violence
- Human Trafficking
